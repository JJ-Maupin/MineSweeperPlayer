This file contains python source code for a basic Minesweeper Strategy, 
as well as detection fo the state of the board.
This uses the website https://minesweeperonline.com/ to be run.

For details about the strategy this proogram is based on, see: 
  http://www.cs.toronto.edu/~cvs/minesweeper/


To compile, using python3: py MineSweep.py 
or using pyinstaller: pyinstaller --onefile MineSweep.py

Executable MineSweep is also Provided.
